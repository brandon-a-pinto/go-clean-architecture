# Go Clean Architecture
Go (Golang) Clean Architecture API project with Chi, PostgreSQL, Docker and more.

## Usage
```bash
git clone https://github.com/brandon-a-pinto/go-clean-architecture.git
cd go-clean-architecture
make prepare
sudo make up_build
```

## Tech
- Golang
- Chi
- gRPC
- GraphQL
- PostgreSQL
- Docker
- Swagger
- Bcrypt
- UUID
- Testify
- Faker
